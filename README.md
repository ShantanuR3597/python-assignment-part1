# Python Assignment - Part 1: Student Grade Tracker

This project is part of an academic assignment focused on Python basics and control flow.

It implements a command-line Student Grade Tracker that:
- Cleans and processes raw student data
- Performs marks analysis
- Generates class performance summaries
- Demonstrates string manipulation techniques

## Features

- Data parsing and cleaning
- Name validation
- Marks analysis using loops and conditionals
- Class performance summary generation
- String manipulation utilities

## Tech Used

- Python 3
- Jupyter Notebook (Google Colab)

## How to Run

1. Open the notebook in Google Colab or Jupyter Notebook
2. Run all cells (Runtime → Run all)
3. Outputs will be displayed below each cell
